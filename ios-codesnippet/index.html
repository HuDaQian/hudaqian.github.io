[iOS]Code Snippet

2020-06-11

iOS开发时，你可以自己使用一些模版来帮助自己少输入一些代码。按住Command + Shift + L 打开Snippet窗口，选择Snippets Library，就能看到当前可以使用的模版了。

当然，也可以自己创建这些模版。
写上模版的关键部分代码，然后右键，选择Create Code Snippet，补充相应信息就行了。

自定义的模版保存的位置是~/Library/Developer/Xcode/UserData/CodeSnippets，当然，只有在自定义了Code Snippet的时候才有这个文件夹。
