期中测试答案解析（20题）

2019-07-14

1、人类的第八大奇迹是什么?（单选）
A、股票
B、指数基金
C、复利
答案：C
解析：称为第八大奇迹的，是复利。复利的神奇之处在于刚开始的时候复利效应是很微小的、不易察觉的，但当发展到一定阶段就会产生非常惊人的效果。不仅仅是投资，我们的学习、工作也会有复利效应哦，每天比别人努力一点点，坚持比别人多一点点，我们能超出其他人一大截。

2、我们需要在什么价格买入什么样的公司？（单选）
A、在股票涨势喜人的时候买入涨的最好的公司
B、在一家会赚钱的好公司跌到低估的时候买入
C、随时买入
答案：B
解析：买股票其实和我们买东西一样，当然要选在它打折比较便宜或者还没被太多人注意到，估值被低估的时候买才最合适，然后坐等它升值后卖出。如果什么也不考虑，不管高低点，盲目买入，之后遇见持续下跌，就有你难受的了。好公司也不能买贵了。

3、关于必要、需要、想要，以下哪个定义是正确的？（单选）
A、想要：改善生活质量的
B、需要：欲望的写照
C、必要：不花会死的
答案：C
解析：在分享和大课中都提到了，必要是没有这件东西你的生活会受到重大影响，不能正常生活，比如我们的一日三餐；需要是对学习成长或者生活有益的但有些超出预算，比如牛奶水过补充身体营养，去健身房健身为了身体健康的；想要仅仅是有了会开心，只是为了满足自己的欲望，比如奢侈品包包，王者荣耀皮肤。

4、债券按照风险从大到小排列争取的是？（单选）
A、国债，地方债，企业债
B、企业债，国债，地方债
C、企业债，地方债，国债
答案:C
解析:债券是一种我们可以直接挑选借款人的投资品。那么他的风险就是按照借款人的偿还能力进行排序。国债，地方债，企业债，三者的借款人分别是国家，地方政府和企业。那如此答案已经显而易见了。当然风险与盈利同在的。他们的收益从大到小也分别是企业债，地方债，国债。

5、铁柱购买了一份死亡寿险，当发生以下哪种情况时，保险公司会给予赔付（单选）
A、检查出血管畸形脑出血，住院微创花了20万
B、跟朋友出去high,摔得半身不遂
C、因为长期熬夜加班，某一天深夜猝死在办公桌上
答案：C
解析：寿险顾名思义，是以人的寿命为保障对象的保险产品，死亡寿险是寿险产品中，被保险人死亡后进行赔付的产品。我们在购买保险产品的时候，一定要明确赔付的条款，什么情况下保险公司会赔什么情况下不赔，不能只听保险代理人讲。

6、消灭拿铁因子第一步是（单选）
A、睡觉
B、记账
C、不喝咖啡
答案：B
解析：“拿铁因子”这个词是由一位叫大卫·巴赫的金融顾问提出的，源于一个故事：一对夫妻，每天早上必定要喝一杯拿铁咖啡，看似很小的花费，30年累积算下来花钱竟达到了70万元。每个人身上可能都存在属于自己的“拿铁因子”，你的拿铁因子不一定是喝咖啡，可能是吸烟，可能是囤只用一两次的化妆品和衣服，总之就是那些我们长期购买但只是为了满足我们欲望的想要品，要消灭“拿铁因子”，首先第一步，就是我们要通过记账分析来找到它，从而在日常生活中进行这方面的支出缩减。

7、记账的时候如果我们发现记账账户上的钱比实际的钱多了，对不上账，而又实在想不起来了钱花在哪里了，以下做法中正确的是：（单选）
A 、花30分钟去仔细的回忆翻找消费记录
B 、想不起来不记了
C 、不跟自己较劲,对不上的差额直接在软件上极为其他支出
答案：C
解析：在记账的过程中，我们经常会遇到一些支出实在想不起来了，这时候我们完全有必要为这几块钱十几块钱想破脑袋，浪费的时间成本才是最贵的，给自己的支出项目设置一项其他支出，对照一下账本原有金额和实际金额，对不上的差额就记其他支出，一点细微的差距不影响我们对整体支出的分析，理财不需要太追求细枝末节了。

8.家庭中优先购买保险的是？(单选）
A、今年25岁，不上班在家的游手好闲的小王
B、今年47岁，目前作为家庭主要收入的爸爸老王
C、今年63岁，退休在家养老的王爷爷
D、今年1岁，可爱美丽又萌萌哒的小宝贝
答案：B
解析：保险优先购买原则是家庭生活的顶梁柱。也就是保障家庭收入的主要经济来源。4个选项明确表示爸爸老王是这个家庭的主要收入保障。而小王虽然年纪轻，但是他没有经济收入，还成为家庭负担。可不能选。

财富自由是什么？（单选）
A、被动收入>支出
B、被动收入<支出
C、被动收入=住处
答案：A
解析：前面已经讲了被动收益的概念。如果你一年的消费是3万，那么你的投资收益只需要3万就能达到初步的财富自由，如果你一年的消费是10万，那么相对应的投资收益要达到10万才能财富自由。有了更多的被动收入，我们就有了更多的选择权，有了对坑爹的老板和坑爹的工作说不的权利。

以下哪些是被动收入？（多选）
A银行的存款利息
B房租的收入
C工资的收入
D投资利息
答案：ABD
解析：这道题也在我们的第一课里面出现咯，所谓的“被动收入”，就是我们不用上班打卡，不用朝九晚五，不用做不喜欢的事、用时间换来的收入，这就是被动收入，常见的被动收入比如：银行存款利息，房租，其他投资收益等等。

11、为什么说理财是人生必备的技能？（多选）
A、能拥有更多财富，让自己更自由；
B、抵抗通货膨胀，不让钱被吃掉；
C、避免被骗被坑，更好养大自己的鹅
答案：ABC
解析：你不理财，财不理你，理财才能获得更多财富，才不用被生计所迫，从事自己不喜欢的工作；通过理财，用投资的收益率抵抗通货膨胀率，可以避免资产贬值悄悄蒸发；学了理财，有了正确的理财思维和投资知识，你就明白投资背后的逻辑，就能分辨出投资陷阱和投资骗局，避免上当，毕竟是我们辛苦赚来的钱，不要给了通胀和骗子。

12、以下哪些是资产？（多选）
A、有股息的股票
B、自住的房子
C、时间
答案：ABC
解析：我们来明确一下资产的概念：资产就是你拥有的，能卖钱、能投资或者自用的东西。负债就是预计能够带来经济利益流出的东西，所以房子是我们的资产。有股息的股票毋庸置疑是资产，因为它能带来净现金流入。我们提过的时间（生命）、劳动能力（脑力和体力）、人力资源这些表外资产也是我们人生中的一大财富，都可以为我们所用置换成现金。如何让我们变得越来越有钱？当然是多买充沛现金流的优质资产，远离负债。

13、保险的双十法则是（多选题）
A、保费占家庭收入的10%
B、保额是家庭收入的10倍
C、保费占家庭收入的10倍
D、保额占家庭收入的10%
答案：AB
解析：双十法则是保费占家庭收入的10%，保费保额是家庭收入的10倍。很多保险代理人会在推销保险的时候讲到双十原则，但实际上，双十原则过于教条主义，一个千万富翁即使配置百分之一的保险也比一个普通工薪阶层拿80%收入配置的要强大很多，年入100万的家庭，每年要拿出10万买保险吗，要配置千万保额吗？我们每个家庭，每个人的经济情况、负债情况和需求都是不一致的，不能一概而论哦。因此，保险是要综合考虑成本与支出才去配置的，这就需要科学的保障体系为支撑，如何根据自己和家庭的情况，给自己设计一个科学的保障体系，在保险实战课里会详细讲到，教大家以自身需求为基础的，以年度现金流入为约束,动态调整，时间关系，这里就不再多说。

14、下列哪些产品你投资后就能成为资产的所有人（多选）
A、股票
B、期货
C、艺术品
答案：BC
解析：投资股票，你就是公司股东，成为部分资产所有人；期货就是一种合约，一种将来必须履行的合约，而不是具体的货物。比如强锅让同学帮忙采购燕窝事件，如果1年后燕窝稀缺了，他能大赚一笔，如果燕窝不让吃了，他的燕窝只能在仓库发霉。艺术品的投资，是你低价买到好的艺术品，最后艺术家出名，你可以高价卖出，但你如果不懂，买了艺术品，就没什么用了。

15、属于投资中介的是（多选）
A、银行
B、P2P
C、信托
答案：ABC
解析:投资中介是指通过资金流获得盈利的中间商。我们将钱借给中间商，获得利益，但是我们不知道中介把我们的钱转借给了谁，对方的偿还能力我们也不知道。遇到坑货中介有可能血本无归……就是一个字惨。

16、以下哪种投资品是低风险、中等风险、高风险都有可能是会涉及到的？（多选）
A、银行理财产品
B、P2P
C、基金
答案：AC
解析：银行理财产品其实风险跨度还是比较大的，从低的到高的都有，所以大家如果以后有投资银行理财产品的话，要注意阅读产品的说明书哦，包括期限、门槛、收益类型、资金投向、风险评级等等信息。
基金也同样，不同种类的基金，从货币基金，到债券基金，混合型基金，股票基金，风险也是逐级递增的。

17、买股票是什么？（多选）
A、买股票就是买公司
B、买股票就是成为公司的股东
C、就是买一堆代码
答案：AB
解析：通过晚间分享我们已经学习到了股票的概念，股票相当于是一家公司的部分资产，我们成为了这家公司的部分资产持有者，所以如何挑选一支优秀的股票的，当然是要先找到一家优秀的，可持续盈利发展的好公司了。

18、买股票赚钱有哪几种方式方式？（多选）
A、低买高卖
B、分红
C、通货膨胀
答案：ABC
解析：低买高卖赚差价割韭菜的钱和优秀上市公司发展盈利带来的股息红利，股息红利又分为现金股利和股票股利，也就是我们说的分红啦。通货膨胀这个人人有份，但是不可预测（水涨船高，通货膨胀，你有一个苹果，本来1块钱，现在2块，账面上赚了1块，如果是持有现金就要贬值）。

19、股票投资四步骤，分别是哪四步？（多选）
A、这家公司是干嘛的？
B、这家公司赚不赚钱？
C、这家公司有没有特殊的优势？
D、这家公司究竟值多少钱？
答案：ABCD
解析：买股票就是买公司，通过四步分析，了解到公司的商业逻辑，判断它的盈利能力，是否可以持续盈利，现在是不是低估值得买入的时刻。

20、让钱生钱的第一步是什么？
A、开源节流
B、学习投资理财知识
C、养大金鹅
答案：开源节流
解析：开源节流人人都懂，但是真正做的到的人有多少，从坚持到习惯是一个漫长的过程，却也是一个收获的过程，哪怕是毫无收入的学生小伙伴，或者负债的小伙伴，相信从每个月的生活费中节省出200元都是妥妥的，以平均10%的年化收益率，坚持投资20年后，你将拥有15万，15万谈不上财富自由，但是也是一笔小小的资金了，更不用说随着年龄的增长，我们的收入也会逐渐的增加，所以小伙伴们一起加油吧！
